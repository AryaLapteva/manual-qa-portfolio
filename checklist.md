## Общие чек-листы

### 1. Чек-лист: Авторизация пользователя (веб-приложение)

**Форма входа:**

- Поле «Логин» отображается
- Поле «Пароль» отображается
- Кнопка «Войти» активна только при заполненных полях

**Позитивные проверки:**

- Вход с корректным логином и паролем
- Вход через Enter
- Вход с сохранёнными данными в Google 

**Негативные проверки:**

- Пустой логин / пароль
- Неверный логин
- Неверный пароль
- SQL-инъекция в поле логина
- Ввод спецсимволов

**UX:**

- Сообщение об ошибке отображается
- Текст ошибки понятный



### 2. Чек-лист: Карточка товара (интернет-магазин)

**Отображение:**

- Название товара отображается
- Цена отображается
- Изображение товара загружается
- Кнопка «Добавить в корзину» видна

**Функционал**

- Добавление товара в корзину
- Изменение количества товара
- Удаление из корзины
- Добавление товара в "Избранное"

**Границы и верстка**

- Кнопки не выходят за блок
- Текст не обрезается
- Изображение не искажено

**Негативные сценарии**

- Добавление товара без авторизации (если требуется)
- Добавление товара при нулевом остатке



### 3. Чек-лист: Поиск по сайту

**Функциональность**

- Поиск по корректному запросу
- Поиск с пустым запросом
- Поиск по частичному совпадению

**Негативные проверки**

- Спецсимволы в поиске
- Очень длинный запрос
- Поиск несуществующего товара

**UX**

- Отображение “ничего не найдено”
- Подсказки при вводе
