# manual-qa-portfolio
QA portfolio with test documentation: test cases, bug reports and checklists

В этом репозитории Вы найдете примеры документации по тестированию, составленные мной.

*android_studio.md* - скриншоты работы в Android Studio для мобильной разработки и тестирования. Создание эмулятора, исследование логов. 

*bug_reports.md* - примеры баг-репортов для учебного сайта Swag Labs.

*checklist.md* - примеры чек-листов на общие темы.
